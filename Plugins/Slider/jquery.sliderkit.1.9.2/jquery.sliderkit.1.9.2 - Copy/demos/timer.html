Timer add-on

Simple photos slideshow

Notes

  • timer option is set to true.
  • autostill option is set to true, so auto-scrolling won't stop when the mouse is going over the slider.
  • If no 'timer' tag is found in the HTML, the script will generate it. But you can also place your own timer bar HTML tag manually in the slider. So you can easily custom the element, adding wrapper around and CSS background images on it.
