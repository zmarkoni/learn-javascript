Methods

In version 0.9.8, plugin methods are implemented.

Right now there are 5 methodes you can call on the plugin:

pause

pause will finish the current animations of a step and then the slider stay idle till you call resume, start or startNextSlide on it

resume

pause will resume the slider at the next step (the one following the pause methode finished)

stop

stop stops the slider at the current point. All running or stacked animations will be set to there end-state. Use start and startNextSlide to start the slider again

start

start will resume the slider at the next step (the one following the stop methode finished)

startNextSlide

startNextSlide will immediately start the next slide

You can call these methods this way:

    $('.slider').fractionSlider('method-name);
